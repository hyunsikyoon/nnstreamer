## How to run Test Cases

- Use the built library as a general gstreamer plugin.

- Unit Test 

For common library unit test
```bash
$ cd build
$ ./unittest_common
```

For all gst-launch-based test cases (mostly golden testing)
```bash
$ cd tests
$ ./testAll.sh
```
